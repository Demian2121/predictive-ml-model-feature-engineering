# # Modelo Predictivo con Ingeniería de Características

## Descripción

Proyecto de Machine Learning enfocado en el desarrollo de un modelo predictivo utilizando datos estructurados. Se aplicaron técnicas de preprocesamiento, ingeniería de características (feature engineering) y validación del modelo para evaluar su desempeño.

## Objetivo

Construir un modelo supervisado capaz de realizar predicciones a partir de un conjunto de datos estructurado, garantizando un flujo completo de preparación, entrenamiento y evaluación.

## Tecnologías utilizadas

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib / Seaborn

## Proceso desarrollado

1. Carga y exploración del dataset.
2. Limpieza y tratamiento de valores faltantes.
3. Codificación de variables categóricas.
4. Feature engineering para mejorar la calidad de las variables.
5. División del dataset en entrenamiento y prueba (Train/Test Split).
6. Entrenamiento de modelo supervisado.
7. Evaluación mediante métricas de desempeño (accuracy, matriz de confusión, etc.).
8. Visualización de resultados.

## Resultados

El modelo fue evaluado utilizando métricas estándar de clasificación, permitiendo analizar su capacidad predictiva y posibles mejoras.

## Conceptos aplicados

- Machine Learning supervisado
- Feature Engineering
- Validación de modelos
- Overfitting y generalización
- Evaluación de desempeño
